# Privacy Policy for HDR Avatar Blocker

**Effective Date:** December 5, 2025

## 1. Data Collection and Usage
**We do not collect, store, use, or share any user data.**

*   **No Tracking:** The extension does not track your browsing history, search queries, or usage patterns.
*   **No Analytics:** We do not use any third-party analytics tools (e.g., Google Analytics) or crash reporters.
*   **No Accounts:** The extension functions entirely without user accounts, login, or registration.

## 2. Network Access
The extension **does not connect to the internet**.
*   It creates no network requests.
*   It sends no data to any external servers.
*   All image processing and logic execution happen locally within your browser.

## 3. Permissions Detail
*   **Storage (`storage`):** This permission is used solely to save your preferred "Image Size Threshold" setting locally. If you have Chrome Sync enabled, this setting allows Google to sync your preference across your signed-in devices. The developer has no access to this data.
*   **Read and modify all your data on the websites you visit:** This standard warning appears because the extension uses "Content Scripts" to detect and modify the CSS style of images on webpages (to fix the HDR brightness issue). The extension **does not** read, store, or transmit the content of the pages you visit.

## 4. Open Source Verification
For complete transparency and security, this project is 100% open source. You can inspect the full source code to verify that no data is being collected or transmitted:

**Source Code Repository:** [https://github.com/heruoxin/hdr-avatar-blocker](https://github.com/heruoxin/hdr-avatar-blocker)

# 隐私政策 (Privacy Policy - Chinese)

**生效日期：** 2025年12月5日

## 1. 数据收集与使用
**我们不收集、不存储、不使用，也不分享任何用户数据。**

*   **无追踪**：本扩展不会追踪您的浏览历史、搜索记录或使用习惯。
*   **无分析**：我们不使用任何第三方分析工具（如 Google Analytics）或崩溃报告工具。
*   **无账户**：本扩展无需注册或登录即可完整使用。

## 2. 网络访问
本扩展**不连接互联网**。
*   它不会发起任何网络请求。
*   它不会向任何外部服务器发送数据。
*   所有的图片检测与处理逻辑完全在您的浏览器本地进行。

## 3. 权限说明
*   **存储 (`storage`)**：此权限仅用于保存您的本地设置（即您设定的“图片尺寸阈值”）。如果您开启了 Chrome 同步功能，该设置会通过 Google 的基础设施在您的设备间同步，但开发者无法访问该数据。
*   **读取和更改您访问的网站上的所有数据**：这是 Chrome 的标准提示，因为本扩展需要使用“内容脚本”来修改网页上图片的 CSS 样式（以修复 HDR 过曝问题）。本扩展**不会**读取、存储或传输您访问的网页内容。

## 4. 开源透明
为了确保绝对的透明与安全，本项目完全开源。您可以随时查阅完整源代码以验证我们没有收集任何数据：

**源代码仓库：** [https://github.com/heruoxin/hdr-avatar-blocker](https://github.com/heruoxin/hdr-avatar-blocker)
