# Chrome 商店描述 (中文)

```text
别让 HDR 显示器上的头像闪瞎你的眼。

如果你用的是 HDR 显示器，肯定遇到过这种情况：本来网页背景是暗的，结果某个人的头像、或者一个小的图标突然亮得刺眼，像个灯泡一样。

这个插件就是用来解决这个问题的。它能把这些刺眼的小图“压”回正常的亮度，让它们看起来和页面其他部分一样舒服。

它能做什么：
• 自动变暗：自动把网页上的小图片（比如头像、Logo）亮度降下来，不再刺眼。
• 简单好用：默认只处理 160x160 像素以下的小图。你也可以自己拖动滑块调整这个大小。
• 安全开源：所有代码都在你电脑上运行，不联网，不收集任何信息。项目已开源：https://github.com/heruoxin/hdr-avatar-blocker

安装完刷新标签页即生效。

提示：如果你在插件菜单里看“Before”和“After”两张图没区别，说明你的显示器不支持 HDR，那你就不需要装这个插件。
```
